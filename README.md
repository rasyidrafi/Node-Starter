# Node Starter
Simple Starter to Connect Frontend and Backend Web Development with NodeJs  

## Installation
Use the package manager NPM to install
```javascript
npm install
```
## Usage

```javascript
npm start
```  
## Features 
- Parsing Variables between Back and Front
- Public Folder and Routes
- Just copy the project and open it, read the code and u will understand
